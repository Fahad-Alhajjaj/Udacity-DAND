Python 2.7 was used for this project.
DAND enviorment was used for this project.

The file p3_main.py is the only file to be run:
python p3_main.py

All files are linked together and p3_main.py is going to call other files and the osm file is read and processed. A JSON file is written after process is completed.

All files have to be in one place, list of files:
p3_main.py
preparing_for_database_mongodb.py
check_streets_types.py
check_postcodes.py
brooklyn_new-york.osm
zip_code_database.csv

and the file 'brooklyn_new-york.osm.json' is produced.
